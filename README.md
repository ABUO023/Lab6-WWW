# Devoir 3 – CSI3540

This project implements a **DOM-based interactive To-Do List application** using JavaScript.

Users can dynamically **add, edit, and delete tasks** directly in the browser.

## Live Demo

The application is hosted at: [https://devoir3.adityabaindur.dev](https://devoir3.adityabaindur.dev)

## Features

- Add new tasks
- Modify existing tasks
- Delete tasks
- Dynamic DOM manipulation using JavaScript
- Simple and responsive UI

## Technologies Used

- HTML
- CSS (Tailwind)
- JavaScript (DOM Manipulation)
