# Laboratoire 6 – CSI3540

A simple task management application built with HTML, CSS, and JavaScript.

## Live Demo

[https://abuo023.github.io/Lab6-WWW/](https://abuo023.github.io/Lab6-WWW/)

## Features

- Add, delete, and reorder tasks
- Adjustable font size
- Print functionality
- Clean, minimal interface

## Technologies

- HTML5
- Tailwind CSS
- JavaScript (DOM Manipulation)
- Lucide Icons
